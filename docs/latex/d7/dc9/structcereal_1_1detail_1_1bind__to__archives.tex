struct cereal\+::detail\+::bind\+\_\+to\+\_\+archives$<$ T, Tag $>$}
Begins the binding process of a type to all registered archives. 

Archives need to be registered prior to this struct being instantiated via the CEREAL\+\_\+\+REGISTER\+\_\+\+ARCHIVE macro. Overload resolution will then force several static objects to be made that allow us to bind together all registered archive types with the parameter type T.
