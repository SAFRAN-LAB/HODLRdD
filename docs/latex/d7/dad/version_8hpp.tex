\doxysubsection{Detailed Description}
Macros to detect cereal version. 

These macros can assist in determining the version of cereal. Be warned that cereal is not guaranteed to be compatible across different versions. For more information on releases of cereal, see \href{https://github.com/USCiLab/cereal/releases}{\texttt{https\+://github.\+com/\+USCi\+Lab/cereal/releases}}.
