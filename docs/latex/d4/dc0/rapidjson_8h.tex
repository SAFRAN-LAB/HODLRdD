Size type (for string lengths, array sizes, etc.) 

Rapid\+JSON uses 32-\/bit array/string indices even on 64-\/bit platforms, instead of using {\ttfamily size\+\_\+t}. Users may override the Size\+Type by defining CEREAL\+\_\+\+RAPIDJSON\+\_\+\+NO\+\_\+\+SIZETYPEDEFINE.
