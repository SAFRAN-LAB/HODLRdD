\subsubsection*{template$<$class Archive$>$\newline
struct cereal\+::detail\+::\+Input\+Binding\+Map$<$ Archive $>$}
A structure holding a map from type name strings to input serializer functions. 

A static object of this map should be created for each registered archive type, containing entries for every registered type that describe how to properly cast the type to its real type in polymorphic scenarios for shared\+\_\+ptr, weak\+\_\+ptr, and unique\+\_\+ptr.
