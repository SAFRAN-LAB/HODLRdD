Extracts the true type from something possibly wrapped in a cereal No\+Convert. 

Internally cereal uses some wrapper classes to test the validity of non-\/member minimal load and save functions. This can interfere with user type traits on templated load and save minimal functions. To get to the correct underlying type, users should use \doxylink{structcereal_1_1traits_1_1strip__minimal}{strip\+\_\+minimal} when performing any enable\+\_\+if type type trait checks.
