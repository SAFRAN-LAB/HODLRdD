\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{classrapidjson_1_1_allocator}{Allocator}} = Crt\+Allocator$>$\newline
struct Generic\+Memory\+Buffer$<$ Allocator $>$}
Represents an in-\/memory output byte stream. 

This class is mainly for being wrapped by \doxylink{class_encoded_output_stream}{Encoded\+Output\+Stream} or \doxylink{class_auto_u_t_f_output_stream}{Auto\+UTFOutput\+Stream}.

It is similar to File\+Write\+Buffer but the destination is an in-\/memory buffer instead of a file.

Differences between Memory\+Buffer and String\+Buffer\+:
\begin{DoxyEnumerate}
\item String\+Buffer has \doxylink{classrapidjson_1_1_encoding}{Encoding} but Memory\+Buffer is only a byte buffer.
\item String\+Buffer\+::\+Get\+String() returns a null-\/terminated string. Memory\+Buffer\+::\+Get\+Buffer() returns a buffer without terminator.
\end{DoxyEnumerate}
