class Generic\+Array$<$ Const, Value\+T $>$}
Helper class for accessing \doxylink{document_8h_a071cf97155ba72ac9a1fc4ad7e63d481}{Value} of array type. 

Instance of this helper class is obtained by {\ttfamily Generic\+Value\+::\+Get\+Array()}. In addition to all APIs for array type, it provides range-\/based for loop if {\ttfamily CEREAL\+\_\+\+RAPIDJSON\+\_\+\+HAS\+\_\+\+CXX11\+\_\+\+RANGE\+\_\+\+FOR=1}.
