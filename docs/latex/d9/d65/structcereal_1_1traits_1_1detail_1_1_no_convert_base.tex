\doxysubsection{Detailed Description}
Used to help strip away conversion wrappers. 

If someone writes a non-\/member load/save minimal function that accepts its parameter as some generic template type and needs to perform trait checks on that type, our No\+Convert wrappers will interfere with this. Using the struct strip\+\_\+minmal, users can strip away our wrappers to get to the underlying type, allowing traits to work properly
