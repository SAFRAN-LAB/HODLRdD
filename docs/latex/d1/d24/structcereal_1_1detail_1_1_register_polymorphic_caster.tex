struct cereal\+::detail\+::\+Register\+Polymorphic\+Caster$<$ Base, Derived $>$}
Registers a polymorphic casting relation between a Base and Derived type. 

Forward declaration, see \doxylink{polymorphic__impl_8hpp}{polymorphic\+\_\+impl.\+hpp} for more information.

Registering a relation allows cereal to properly cast between the two types given runtime type information and void pointers.

Registration happens automatically via \doxylink{structcereal_1_1base__class}{cereal\+::base\+\_\+class} and \doxylink{structcereal_1_1virtual__base__class}{cereal\+::virtual\+\_\+base\+\_\+class} instantiations. For cases where neither is called, see the CEREAL\+\_\+\+REGISTER\+\_\+\+POLYMORPHIC\+\_\+\+RELATION macro
